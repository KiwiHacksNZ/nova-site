# Contributing to the KiwiHacks website

## Getting set up

1. Clone it (duh)
2. Install dependencies with bun (`bun install`)
3. Run the dev server (`bun dev`)

## Working on site

1. Make a branch
2. Code (duh)
3. Before pushing format (`bun format`)
4. Push and make a PR
5. Ping Kyle, Josh, or Sebastian in bts for review
